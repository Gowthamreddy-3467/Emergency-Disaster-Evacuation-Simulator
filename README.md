# 📌 Project Overview

**EvacuAid** is a Java Swing desktop application developed as a **Data Structures and Algorithms (DSA) Mini Project**.

The system simulates emergency evacuation during disasters by modelling an entire city as a **Graph** where:

- 📍 Locations are represented as **Nodes**
- 🛣 Roads are represented as **Edges**
- 🚑 Dijkstra's Algorithm finds the safest evacuation route
- 🌪 Disaster simulation affects road availability and traffic
- 🚒 Priority Queue manages emergency rescue requests

The project demonstrates how classical Data Structures and Algorithms can solve real-world emergency management problems.
