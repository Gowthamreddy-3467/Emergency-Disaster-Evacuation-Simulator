# 🚨 EvacuAid – Smart Disaster Response & Emergency Evacuation Management System

## 📌 Project Overview

EvacuAid is a Java-based Emergency Disaster Evacuation Simulator that demonstrates the real-world application of Data Structures and Algorithms (DSA).

The project models a city as a graph, where:

- Locations are represented as Nodes.
- Roads are represented as Edges.
- Graph algorithms are used to compute the best evacuation routes during disasters.

The goal is to help emergency responders and civilians find safe evacuation paths during situations such as floods, fires, earthquakes, cyclones, and chemical leaks.

---

## 🎯 Project Objectives

- Apply Graph Data Structures to a real-world problem.
- Implement shortest path algorithms for evacuation planning.
- Simulate disaster scenarios.
- Build a modular Java application using Object-Oriented Programming.
- Demonstrate practical applications of DSA concepts.
